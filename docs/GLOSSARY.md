# Glossary

## Sivraj

The product: a persistent intelligence and identity layer for the AI-native world.

## Twin

The user's evolving cognitive model. It contains memory, graph structure, synthesis, preferences, goals, and permissions.

## Memory Fragment

A retrievable unit of context extracted from a source artifact.

## Source Artifact

An uploaded or connected source such as a PDF, note, chat export, GitHub repo, voice note, calendar item, or email.

## Cognitive Graph

The living identity graph that tracks people, goals, projects, beliefs, workflows, habits, decisions, expertise, and emotional patterns.

## Context Packet

A scoped bundle of relevant context returned to an AI app or agent.

## Synthesis Engine

The system that turns raw memory into strategic insight, pattern detection, reflection, and recommendations.

## Reflection

A periodic or triggered synthesis that updates the user's understanding of goals, behavior, drift, and opportunities.

## Walrus

Persistent cognitive infrastructure used for memory blobs, archives, agent histories, reasoning traces, and long-term identity state.

## MemWal

Semantic retrieval and memory ranking layer used for long-term recall and context selection.

## Seal

Encryption and scoped access layer for private memory and confidential agent coordination.

## Sui

Identity and programmable ownership layer for user-controlled memory and access rights.
